# Back end - wegodevform

## Build Setup

```bash
# install dependencies
$ npm install

# di localhost jalankan
$ npm run dev

# di production server jalankan ini
$ npm run build
$ npm run start
```

## Deskripsi

### `env file`

Rename `sample.env` menjadi `.env`, kemudian sesuaikan settingan sesuai dengan pc/server kamu
